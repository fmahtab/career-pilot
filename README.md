# CareerPilot

CareerPilot is an AI-powered job application management platform built with FastAPI and PostgreSQL.

## Features

### Authentication

* User registration
* Secure password hashing
* JWT-based authentication

### Job Application Management

* Create applications
* View applications
* Update application status
* Delete applications

### Technology Stack

* FastAPI
* PostgreSQL
* SQLAlchemy
* JWT Authentication
* Python

## Getting Started

1. Clone the repository
2. Create a virtual environment
3. Install dependencies
4. Configure PostgreSQL
5. Run:

```bash
uvicorn app.main:app --reload
```
